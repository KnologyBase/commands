    ErrorLog logs/dummy-host.example.com-error_log
    CustomLog logs/dummy-host.example.com-access_log common
</VirtualHost>

Apache Logs 

Categories 

200- good
300 - redirects
400 - client-invoked errors
500 - server-invoked errors


Log Format 

%h - host (IP)
%l - IDENT (deprecated)
%u - user
%t - timestamp
%r - request method ( Get/POST)	
%s - status	(2xx, 3xx ,4xx)
